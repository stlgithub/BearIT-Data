About the Project

A project in which we created a local and cloud based database for a fictional Christmas Pop-Up Shop. The databases include data about customers, products, and sales, and the data was used to create various reports. Scrum methodologies were used during the project, and the duration of the project was two sprints, or four weeks. Project was a part of BearIT's ICT-Camp.
